## 2. 핵심 발견: 사업계획서가 실제 구현을 과소 표현하고 있다

Architect 분석 결과, **MVP의 70-80%가 이미 구현 완료**되어 있다:

| 구성요소 | 완성도 | 근거 |
|---------|:------:|------|
| DB 스키마 + 마이그레이션 | 95% | 20개 마이그레이션, RLS 12테이블 |
| 추천엔진 | 90% | JSON DSL 룰트리, 14개 연산자, 6요소 스코어링, 테스트 6건 |
| 데이터 파이프라인 | 85% | 7개 소스 코드 완성 (온통청년/행안부/워크넷/장학재단/드림스폰/금감원/지역포털) |
| API 서버 | 85% | 13개 라우트 모듈, JWT 인증/인가 완성 |
| 모바일 앱 | 80% | 20+ 화면, Moss Stone 디자인 시스템 |
| 웹 관리자 | 75% | CRUD + 동기화 + 통계 |
| 알림 시스템 | 80% | 3채널 디스패처 (Push/FCM/카카오 알림톡) |

**코드량**: Rust 7,935줄 + Mobile 35,636줄 + Web 7,344줄 + SQL 마이그레이션 20개 = 약 50,000줄

이것을 사업계획서에 반영하면 **실현가능성 점수가 6 → 9로** 대폭 상승한다.

## 3. 심사위원 취약점 TOP 5 + 대응 방안

### 취약점 1: 숫자가 없다 (Codex 3/10)
**현재**: TAM/SAM/SOM "실제 수치 입력 필요"로 빈칸
**대응**: 아래 수치 삽입
- TAM: 전국 19-34세 청년 약 927만 명 (통계청 2025)
- SAM: 부산·대구 19-29세 대학생/휴학생/취준생 약 38만 명
- SOM: 출시 1년차 부산 소재 대학 10곳 재학생 중 침투율 5% = 약 8,500명
- KPI 목표: 온보딩 완료율 65%, 추천 조회율 80%, 북마크율 35%, 신청처 클릭률 25%, 7일 재방문율 30%

### 취약점 2: BM이 검증되지 않았다
**현재**: PRO 유료화 + 제휴 수수료 (구조만 언급)
**대응**:
- PRO 월 3,900원 x 전환율 3-5% = 월 MAU 8,500명 기준 월 100-170만원
- 제휴 수수료: 금융/교육/주거 파트너 CPA 건당 3,000-10,000원
- 삼쩜삼 사례 인용 (세금 환급 CPA 모델로 MAU 1,300만 달성)

### 취약점 3: 경쟁우위가 선언적이다

**현재**: 역할만 나열, 실명/경력/포트폴리오 없음
**대응**:
- 대표자 프로필 + 관련 경험 (정책 탐색 실제 경험, 개발 역량)

**현재**: 추천, 알림, 상태추적, 서류보관함, 자동화, 원클릭까지 나열

- 1차 (협약기간): 추천 + 자격체크 + 북마크 + 알림 + 체크리스트 (이미 구현 완료)
- 2차: PRO 자동입력 + 제휴 간편신청
- 3차: 지역 확장 + AI 고도화

## 5. 섹션별 보강 권고

### Problem 보강 (7→9)
1. **정량 수치 추가**: "2024 청년의 삶 실태조사에 따르면 청년 정책 인지율은 37.5%에 불과하며, 매년 약 150만원의 수혜 기회를 놓치고 있다"
2. **Before/After 시나리오**: "매일 3시간씩 5개 사이트를 뒤지다 서류 미비로 탈락한 A씨" vs "마지미 알림 하나로 5분 만에 300만원 장학금을 수령한 B씨"
3. **외적동기 2 + 내적동기 1 구조 명확화**:
   - 외적동기 ①: 정보 파편화 (2,000+ 정책이 10+ 사이트에 분산)
   - 외적동기 ②: 신청 준비 부담 (신청 포기율 추정 40%+)
   - 내적동기: 행동 유도 실패 ("나와 상관없을 것 같다" 인지 장벽)

### Solution 보강 (6→9)
1. **한 것 vs 할 것 명확 분리**:
   - 한 것: 추천엔진 구현(14개 연산자, 6요소 스코어링), 7개 데이터 소스 파이프라인, 20+ 화면 모바일 앱, 13개 API 라우트, JWT 인증, RLS 보안, DB 마이그레이션 20건, 통합 테스트 10건
   - 할 것: 카카오 로그인 실연동, 공공 API 키 확보 + 실데이터 적재, 베타 테스트
2. **개발 프로세스 이미지 필수**: 시스템 아키텍처 다이어그램, 데이터 수집-정규화-추천 플로우차트
3. **Rust 선택 이유 1줄**: "복잡한 룰 평가에서 런타임 에러를 원천 차단하는 타입 안전성과 컴파일타임 쿼리 검증"

### Scale-up 보강 (5→8)
1. **TAM/SAM/SOM 수치 + 산식 + 출처** 삽입
2. **경쟁사 비교표** (위 표 참조)
3. **성과지표 Big4**: 온보딩 완료율 65%, 추천 조회율 80%, 신청처 클릭률 25%, 7일 재방문율 30%
4. **매출 추정**: 1년차 MAU 8,500 x PRO 3% x 월 3,900원 = 연 1,190만원 + 제휴 CPA
5. **ESG**: "왜 부산·대구인가?" → 전국 청년 유출 최심각 지역으로 지자체 청년 예산 투입이 가장 공격적

### Team 보강 (5→7)
1. **대표자 실명 + 관련 경력** (정책 탐색 실경험, 개발 이력)
2. **기술 역량 증빙**: 이미 Rust 7,935줄 + Mobile 35,636줄 + Web 7,344줄 구현 완료
3. **고용 계획**: 데이터 검수 1명, 마케팅 1명 (협약기간 내)
4. **협력기관**: 부산시 청년정책과, 대학 학생처, 한국장학재단 (연계 추진 중)
