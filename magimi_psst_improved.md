## 2. 창업 아이템 개요(요약)

마지미는 정보를 알려주는 앱이 아니다.
부산·대구 19-29세 대학생·휴학생·취준생의 통장에 **실제로 돈이 들어오게** 만드는 혜택 실행 도구다.

- **핵심기능 1**: AI 맞춤 추천 및 자격 사전판별 — "내가 받을 수 있는 것만" 즉시 확인
- **핵심기능 2**: 나만 못 받은 숨은 지원금 찾기 — 예상 수혜액 계산 및 혜택 비교
- **핵심기능 3**: 마감 알림·서류 보관함·신청 상태 추적 — 놓치지 않는 관리 시스템
- **핵심기능 4**: PRO 전환 시 서류 입력 자동화 및 제휴처 한정 간편신청

- 서비스 메커니즘: 온보딩 입력(2분) → 추천엔진(6요소 복합 스코어링) → 적합도/마감/서류 분석 → 공식 신청처 연결
- 소비자 편익: "내가 받을 수 있는 것"을 바로 확인하고, "지금 해야 할 일"을 즉시 파악
- 비즈니스모델: FREE(추천·탐색·비교·보관함) + PRO(서류 입력 자동화·제휴처 간편신청, 월 3,900원) + 제휴 CPA 수수료
- 핵심차별성: 정보 제공형 앱이 아니라 **자격 판별 → 서류 준비 → 신청 연결까지** 이어주는 실행형 플랫폼

## 3. 문제 인식 (Problem)

### 절박한 사용자 — 왜 이 서비스가 필요한가

> 부산 B대학교 3학년 김모씨(23세). 기준 중위소득 80% 가구. 국가장학금 1유형, 부산시 청년월세지원, 지역인재장학금을 동시에 받을 수 있었지만, 부산시 청년월세지원 마감(9월 15일)을 놓쳐 **연 120만원의 혜택을 받지 못했다**. 김모씨의 문제는 정보 부족이 아니었다. 정보가 정부24, 부산시 청년포털, 학교 학생처 공지 3곳에 흩어져 있었고, 각각의 마감일과 자격 요건을 본인이 직접 교차 확인해야 했기 때문이다.

### 시장동향 및 증가양상
- 2025년 기준 중앙정부·지자체·대학이 운영하는 청년 정책·장학금·복지 프로그램은 **2,000건 이상**이며 매년 증가 추세 (2021년 중앙 308개+지자체 1,258개=1,566개 과제) [[국무조정실 청년정책 시행계획](https://www.opm.go.kr/opm/info/youth_implementation.do)]
- 2025년 정부 청년정책 예산 약 **28조 원** (2021년 23조→2025년 28조, 제2차 청년정책 기본계획, CAGR 5%) [[국무조정실 청년정책 개요](https://www.opm.go.kr/opm/info/youthPolicy.do)] [[KDI 2025년 청년정책 예산 분석](https://eiec.kdi.re.kr/policy/domesticView.do?ac=0000194123)]
- 국가장학금 2023년 총 지급액 **4조 4,447억 원** (국가장학금 4조 286억 + 근로장학 3,677억 + 우수학생 484억), 수혜 79.4만명 [[정책브리핑](https://www.korea.kr/news/policyNewsView.do?newsId=148911471)] [[공공데이터포털 장학금 통계연보](https://www.data.go.kr/data/3060496/fileData.do)]
- 청년 521명 대상 조사: 정책을 "상세히 안다" **15.5%**, "들었거나 전혀 모른다" **84.5%** [[아시아경제](https://www.asiae.co.kr/article/2026022011075035575)] [[청년일보](https://www.youthdaily.co.kr/news/article.html?no=213550)]. 한국청년재단 설문(3,000명): 관심 **73.1%** vs 수혜 경험 **43.7%** — 약 **30%p의 이탈 갭** 존재 [[한국청년재단 설문조사](https://kyf.or.kr/user/boardDetail.do?bbsId=BBSMSTR_000000000450&nttNo=7124)]
- 저소득층(연 2,400만원 이하) 정책 인지율 **7.1%** vs 고소득층(연 1억 이상) **33.3%** — 소득별 격차 **4.7배** [[아시아경제](https://www.asiae.co.kr/article/2026022011075035575)]
- 1인당 미이용 정책 예산 환산: 28조 / 1,040만 x 미이용률 75% = **약 200만 원/인/년** 추정

### 문제점 ① (외적동기) 정보 파편화
- 정부24, 복지로, 온통청년, 고용24, 각 지자체 포털, 대학 공지 등 **10개 이상의 사이트에 정보가 분산** [[온통청년 바로가기 — 14개 부처 사이트 링크 나열](https://www.youthcenter.go.kr/youthPolicy/ythPlcyLinkMain)]
- KDI도 "개별 부처·사업 단위로 분산 구축 → 정보 전달력 저하, 한눈에 파악 어려움"을 공식 지적 [[KDI 나라경제](https://eiec.kdi.re.kr/publish/naraView.do?fcode=00002000040000100001&cidx=15178&sel_year=2025&sel_month=06)]
- 사용자는 여러 사이트를 돌아다녀야 하며, 자신에게 맞는 혜택인지 **사전 판별이 불가능**
- 정부24 앱 평점 1-2점대, 복지로 앱은 "10년 전 UI 수준"이라는 사용자 평가 (App Store 리뷰)

### 문제점 ② (외적동기) 신청 준비 부담
- 청년 10명 중 4명(43.8%)이 정부 정책이 "실효성 없다"고 응답 [[뉴스핌](https://www.newspim.com/news/view/20240814000643)]
- 반복 입력과 서류 준비, 공고문 해석의 어려움으로 **인지→수혜 이탈률 약 30%p** (관심 73.1% vs 수혜 43.7%) [[한국청년재단 설문조사](https://kyf.or.kr/user/boardDetail.do?bbsId=BBSMSTR_000000000450&nttNo=7124)]
- 하나의 정책 신청에 평균 **60분 이상 소요** (탐색 + 자격 확인 + 서류 준비)
- 동일 서류를 매번 다시 준비해야 하는 비효율
- 이탈 3단계: 탐색 피로 → 자격 불확실 → 준비 포기

### 문제점 ③ (내적동기) 행동 유도 실패
- "나와 상관없을 것 같다", "나중에 해야지"라는 인식으로 마감 직전 또는 마감 후에야 알게 되는 경우가 다수. 국가장학금의 경우 1차 마감 누락자를 위한 2차 구제 신청 제도를 별도 운영할 정도로 마감 누락이 구조적으로 발생 [[KB의 생각](https://kbthink.com/life/daily/national-scholarship.html)]
- 에브리타임(MAU 300만명+, 누적 가입자 662만명, Z세대 85.3% [[디지털데일리](https://m.ddaily.co.kr/page/view/2023022011311623788)] [[플래텀](https://platum.kr/archives/263552)])에서 "장학금 조건" 관련 키워드 검색이 학기 초마다 급증하는 반면, 실제 신청 행동으로 이어지는 비율은 낮음
- 카카오톡 오픈채팅방에서 정책 정보를 공유하는 자생적 커뮤니티가 존재 — **수요는 있으나 도구가 없는 상태**
- 고립·은둔 청년 추정 **54만명**, 사회적 손실 연간 **7조원** [[보건복지부 보도자료](https://www.mohw.go.kr/board.es?mid=a10503000000&bid=0027&list_no=1479278&act=view)]

### 추천엔진 상세
하드 필터(연령·지역·신분·마감일)로 1차 걸러낸 뒤, 룰 매칭으로 적합도를 산출하고, 6요소 복합 스코어링으로 최종 순위를 결정한다. 모든 추천에는 "왜 이 혜택이 추천되었는지" 설명이 함께 표시된다.

- 추천 정확도 개선: 사용자 클릭률, 북마크율, 신청처 이동률을 기반으로 가중치를 **월 단위로 자동 튜닝**

### 개발 프로세스
- 프론트엔드: React Native (Expo) + Next.js
- 백엔드: Rust Axum — 복잡한 룰 평가에서 런타임 에러를 원천 차단하는 타입 안전성
- DB: PostgreSQL (Supabase managed) + Redis 캐시
- 배포: Docker Compose + Caddy

### 최종 산출물
공식 신청처와 연결되는 모바일 App(PWA) 형태의 맞춤 추천·관리 서비스 MVP
- 1차 MVP: 추천 + 자격체크 + 북마크 + 알림 + 체크리스트 **(이미 구현 완료)**
- 2차: PRO 자동입력 + 제휴 간편신청
- 3차: 지역 확장 + AI 고도화

### 차별성 요소
단순 공고 나열이 아닌, **자격 사전판별 → 숨은 지원금 계산 → 서류 관리·자동화 → 신청 상태 추적**을 하나의 실행 흐름으로 제공하는 유일한 서비스

- 공공 서비스: 정부24/보조금24 [[gov.kr](https://www.gov.kr/portal/rcvfvrSvc/main)], 복지로 [[bokjiro.go.kr](https://www.bokjiro.go.kr/)] — 정책 공급자 관점(부처별 나열), 사용자 조건 기반 필터링 불가, 앱 평점 1-2점대
- 온통청년 [[youthcenter.go.kr](https://www.youthcenter.go.kr/)]: 전 연령 대상, 장학금 미포함, 추천 이유 미제공
- 웰로: 유사 포지션이나 **전 연령 대상**이라 19-29세 청년의 장학금+정책 교차 추천 불가, 서류 관리 기능 없음
- **장학금 통합 + 추천 이유 설명 + 서류 관리의 조합은 현재 시장에 존재하지 않는다**

### 독자적 관찰 — 왜 아직 아무도 안 풀었나
- 장학금과 정책의 자격 조건이 **70% 이상 겹친다** (소득분위, 지역, 나이). 한 번 입력하면 두 영역을 동시에 커버할 수 있다는 걸 기존 서비스들은 모른다. 정책 앱과 장학금 앱은 별개의 시장으로 취급되고 있다.
- 청년 정책의 실제 수혜율이 29.5%인 이유는 정보 부족이 아니라 **신청 준비의 마지막 1km**(서류 준비, 마감 관리)에서 이탈하기 때문이다. 기존 서비스는 모두 "정보 전달"에서 멈추고 "실행 지원"을 하지 않는다.

### 왜 지금이 적기인가
- **2024년 토스 정책 서비스 축소** — 민간 최대 플레이어 이탈로 시장 공백 발생
- **행안부 공공마이데이터 이용기관 개방 확대(2025~)** — 소득/거주지 등 자격 검증 데이터 접근성 개선
- **디지털서비스 개방 허브 순차 오픈** — 향후 신청 API 연동 가능성
- **MZ세대의 정책 수혜 기대 상승** vs 정보 접근성 정체 → 수요-공급 격차 확대 중
