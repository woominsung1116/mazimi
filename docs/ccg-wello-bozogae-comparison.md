# 마지미 vs Wello vs 보조개 경쟁 분석 (CCG 합성)
> 2026-03-21 | Claude + Codex(GPT-5.4) + Gemini 웹 검색 기반 실제 조사

## 경쟁 앱 팩트체크

### Wello (웰로)
- **범위**: 전국 (부산 전용 아님). "5,200만 국민을 위한 개인 맞춤형 정책 추천 서비스"
- **사용자**: 220만명, 시리즈A 투자 유치
- **데이터**: 전국 2,700개 행정기관/지자체, 일 1만건 이상 업데이트
- **추천**: 10가지 프로필 필터 기반 매칭
- **기능**: 맞춤 정책 추천, 통합 검색, 정책 콘텐츠, 가족 프로필, 알림, 고향사랑기부, 웰로비즈(B2B)
- **평점**: App Store 3.4/5 (45개)
- **약점**: 신청 가능성 % 없음, 서류 준비 가이드 부족, 정보 탐색 중심(실행 약함), 잦은 버그

### 보조개
- **범위**: 전국 (소상공인/사업자 중심, 2026.02부터 개인 보조금 추가)
- **사용자**: 초기 단계
- **추천**: AI 매칭률 표시 + 추천 사유 제시
- **기능**: AI 요약, 맞춤 보조금 추천, 마감 알림, 챗봇
- **평점**: App Store 5.0/5 (14개, 표본 작음)
- **약점**: 서류 준비 없음, 금융 연계 없음, 데이터 소스 불투명, 개인 기능 초기

## 마지미만의 킬러 피처 Top 7

1. **혜택 중복 계산기** - A+B 동시 수혜 가능 여부 + 최대 수령액 설계
2. **서류 발급 지점 맵** - 가까운 무인발급기/주민센터 위치 안내
3. **원클릭 조건 시뮬레이터** - 이사/취업/소득 변화 시 새 혜택 즉시 확인
4. **Time-to-Money 지표** - "이 서류 떼면 시급 50만원짜리 행동"
5. **지역 이동 비교** - "부산→대구 이사하면 혜택 얼마나 바뀜?"
6. **창업 교량 기능** - 개인 혜택 → 창업 시 기업 혜택 로드맵
7. **숨은 돈 영수증** - 편의점 영수증 스타일 SNS 공유 이미지

## 투자자 메시지

> Wello는 정책 디스커버리 플랫폼, 보조개는 AI 요약 탐색 앱, 마지미는 그 둘보다 아래 단계까지 내려가는 실행형 인텔리전스 인프라입니다. "더 많이 보여주는" 게 아니라 "실제로 받게 해주는" 앱입니다.
