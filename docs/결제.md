## 결제 순서 (권장)

1. **지금**: 도메인 구매 + VPS 구매 → 배포
2. **앱 완성 후**: Apple Developer + Google Play → 스토어 출시
3. **프리미엄 기능 추가 시**: 사업자등록 → 통신판매업 → PortOne + PG 계약
4. **유저 확보 후**: 알림톡, Supabase, 마케팅
