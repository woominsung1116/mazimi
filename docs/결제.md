## 필수 (서비스 운영)

| 항목 | 가격 | 결제 주기 | 용도 | 링크 |
|------|------|----------|------|------|
| 도메인 (.kr) | ~15,000원 | 연간 | mazimi.kr 서비스 주소 | 가비아, 호스팅KR |
| VPS (Vultr Seoul 등) | $24~48 (3~6만원) | 월간 | Docker Compose 서버 (DB+API+웹+푸시 전부 포함) | vultr.com, lightsail |
| 카카오 개발자 | 무료 | - | OAuth 소셜 로그인 | developers.kakao.com |

**소계: 월 3~6만원 + 연 1.5만원**

> VPS 하나에 PostgreSQL, Redis, API 서버, 웹, 워커, 푸시 알림 전부 포함.
> 앱 푸시 알림은 Expo Push (무료) 사용. 카카오 알림톡 불필요.

| 항목 | 가격 | 결제 주기 | 용도 | 링크 |
|------|------|----------|------|------|
| Apple Developer Program | $99 (~13만원) | 연간 | iOS 앱스토어 출시 | developer.apple.com |
| Google Play Developer | $25 (~3.3만원) | 1회 | Play스토어 출시 | play.google.com/console |

**소계: 1회 16만원 + 연 13만원 갱신**

## 결제 순서 (권장)

1. **지금**: 도메인 구매 + VPS 구매 → 배포
2. **앱 완성 후**: Apple Developer + Google Play → EAS Build → 스토어 출시
3. **프리미엄 기능 추가 시**: 사업자등록 → 통신판매업 → PortOne + PG 계약
4. **유저 확보 후**: VPS 업그레이드, 마케팅
