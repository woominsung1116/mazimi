# 마지미 전체 기능 흐름 — 사용자 스토리

> 부산 대학생 김지수(24세)가 마지미를 설치하고, 정책/장학금을 찾고, 실제로 신청 완료하기까지의 전체 흐름.

---

## 등장인물

**김지수** (24세, 부산대 경영학과 3학년, 자취, 취업 준비 중)

---

## 1. 온보딩 (첫 실행)

지수가 에브리타임에서 "부산 대학생 장학금 정리" 글을 보고 마지미를 설치합니다.

```
앱 실행 → 온보딩 시작

Step 1/3: "어느 지역에 거주하시나요?"
  → [부산] 선택 (파란 테두리)

Step 2/3: 나이 "24" 입력 + 대학상태 [재학] 선택

Step 3/3: 취업상태 → [미취업 (구직 중)] 선택

→ [다음] 그라디언트 버튼 탭
```

---

## 2. 놓친 돈 계산기 (동기부여)

온보딩 직후 "놓친 돈 계산기" 화면이 나옵니다.

```
✦ 놓친 돈 계산기

연간 최대 4,320,000원
(숫자가 0에서 카운트업 애니메이션)

카테고리별:
📚 장학금      3건  월 150,000원
🏛 청년 정책   5건  월 100,000원
💼 복지/생활   2건
🏦 금융상품    1건
🏢 기업 혜택   1건

"지금 바로 받을 수 있는 혜택이 이렇게 많아요!"

[내 맞춤 혜택 보러가기 →]
```

지수: "헐 400만원이나? 진짜?" → 홈 화면으로 이동

## 3. 홈 화면에서 탐색

홈 탭에 맞춤 추천 카드가 뜹니다.

```
이번 달 내가 받을 수 있는 지원

[장학금 3건] [부산 청년 정책 5건] [복지/생활 2건]

나에게 맞는 추천:
┌──────────────────────────┐
│ 🏷 추천                   │
│ 📍 부산광역시              │
│ 부산 청년 월세 지원         │
│ 월 20만원                 │
│ #주거지원 #월세            │
└──────────────────────────┘

유용한 도구:
[🧮 혜택 중복 계산기]  [🗺️ 지역 이동 비교]
```

지수는 **"부산 청년 월세 지원"** 카드를 탭합니다.

## 6. 구비서류 탭 — Level 1: 서류 가이드

"구비서류" 탭을 누릅니다.

```
지원내용 │ 선정기준 │ 신청방법 │ [구비서류]

☐ 임대차계약서                [방문필요] 🔴
   집주인에게 요청 필요

☐ 소득금액증명원              [즉시] 🟢
   홈택스(hometax.go.kr)에서 온라인 발급
   [발급하기 →]

☐ 주민등록등본                [즉시] 🟢
   정부24(gov.kr)에서 온라인 발급
   [발급하기 →]

☑ 통장 사본                  [즉시] 🟢  ← 보관함에서 가져옴 📁
```

지수는 "발급하기" 버튼을 눌러 **정부24에서 주민등록등본을 발급**합니다.

## 7. 서류 보관함에 저장 — Level 3: 서류 보관함

발급받은 등본을 서류 보관함에 저장합니다.

```
내 정보 탭 → "서류 관리" 카드
→ 서류 보관함

보관 서류 2건

📄 통장 사본          [유효 ✅]  발급: 2026.03.15
📄 주민등록등본        [유효 ✅]  발급: 2026.03.21

[+] 서류 추가

→ [+] 탭
→ 종류: "소득금액증명원" 선택
→ [📷 카메라] 선택
```

## 9. 사전진단 자동 업데이트

서류 보관함에 등본+소득증명원이 추가됐으니, 상세페이지로 돌아가면 사전진단이 자동 업데이트됩니다.

```
┌─ 신청 사전진단 ──────────────────────┐
│  종합 판정: 신청 가능 ✅    92% 일치  │
│                                      │
│  서류 준비  3/4 완료                  │
│  ██████████░░ 75%                   │
│  ✅ 주민등록등본 — 보관함에서 가져옴 📁 │
│  ✅ 소득금액증명원 — 보관함에서 가져옴 📁│
│  ✅ 통장 사본 — 보관함에서 가져옴 📁   │
│  ⚠️ 임대차계약서 (방문필요)           │
│                                      │
│  마감까지 15일 / 준비 예상 3일        │
│  ✅ 시간 여유 있음                    │
└──────────────────────────────────────┘
```

지수: "임대차계약서만 집주인한테 받으면 끝이네!" → 집주인에게 연락

```

│ 생년월일    2002-03-15         [복사] │
│ 주소       부산광역시 금정구... [복사] │
│ 연락처     010-1234-5678      [복사] │
│                                      │
│            [전체 복사]               │
└──────────────────────────────────────┘

┌─ 학적 정보 ──────────────────────────┐
│ 학교       부산대학교                 │

## 14. 지역 이동 비교

지수가 졸업 후 서울 취업을 고민하며 확인합니다.

```
홈 → "지역 이동 비교" 카드

현재 지역: [부산]  →  이동 지역: [서울]

+2건 더 받을 수 있어요!

잃는 혜택 (부산 전용):
🔴 부산 청년 월세 지원 (월 20만원)
🔴 부산 교통비 지원 (월 5만원)

얻는 혜택 (서울):
🟢 서울시 청년 월세 지원 (월 20만원)
🟢 서울시 청년수당 (월 50만원)
🟢 서울시 청년 교통비 (월 6만원)
🟢 서울 청년 면접수당 (회당 5만원)

유지되는 혜택:
⚪ 국가장학금 (전국)
⚪ 청년도약계좌 (전국)

월 25만원 → 월 76만원 (+51만원 증가!)
```

지수: "서울 가면 오히려 더 많이 받네!"

## 15. 상태 추적 + 알림

2주 후, 지수의 폰에 푸시 알림이 옵니다.

```
🔔 마지미 마감 알림
"국가장학금 1유형 마감 3일 전! 지금 신청하세요"
→ 알림 탭 → 상세페이지로 이동

상태 추적:

## 전체 흐름 요약

```
설치 → 온보딩 → 놓친돈 계산기 → 홈
  ↓
프로그램 카드 탭 → 상세페이지
  ↓
사전진단 확인 → 구비서류 탭 → 발급처 링크로 발급
  ↓
서류 보관함에 스캔 저장 → 사전진단 자동 업데이트
  ↓
"바로 신청하기" → 원클릭 어시스턴트 5단계
  ↓
Step 1: 자격 확인 (92%)
Step 2: 서류 확인 (보관함 자동 연동)
Step 3: 정보 확인 (수정 가능)
Step 4: 웹뷰 + 자동 입력 → 제출은 직접
Step 5: 완료! → 상태 추적 시작
  ↓
혜택 중복 계산기 → "월 50만원 가능!"
지역 이동 비교 → "서울 가면 76만원"
PDF 생성 → 엄마에게 카톡 전송
  ↓
마감 알림 → 다른 프로그램도 신청 → 반복

결과: 지수는 연간 594만원을 받게 됨 🎉
```

## 사용 기능 매핑

| 흐름 단계 | 사용 기능 | Level |
|----------|----------|:-----:|
| 온보딩 | 3단계 설문 + 프로필 저장 | - |
| 놓친 돈 계산기 | 룰엔진 매칭 + 애니메이션 | - |
| 상세페이지 4탭 | 지원내용/선정기준/신청방법/구비서류 | 1-2 |
| 사전진단 | 룰엔진 + 서류 상태 + 마감 분석 | 1-2 |
| 서류 가이드 | 발급처 링크 + 소요시간 + 정렬 | 1 |
| 신청 가이드 | 단계별 가이드 + 딥링크 | 2 |
| 상태 추적 | 6단계 트래커 + QuickAction | 2 |
| 서류 보관함 | 업로드 + 스캔 + 암호화 + 매칭 | 3 |
| 자동 채우기 | 클립보드 + 웹뷰 JS 주입 | 4 |
| PDF 생성 | pdf-lib + 공유 + 인쇄 | 6 |
| 원클릭 어시스턴트 | 5단계 위저드 | 7 |
| 혜택 중복 계산기 | 충돌 분석 + 총 혜택가 + 금융 시뮬 | - |
| 지역 이동 비교 | 지역별 혜택 비교 | - |
| 푸시 알림 | Expo Push + 마감 알림 | - |
